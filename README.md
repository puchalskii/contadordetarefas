# Lista de Tarefas

## Objetivo
Um programa simples para adicionar e listar tarefas.

## Como usar
1. Execute o arquivo `lista_tarefas.py`.
2. Escolha as opções no menu para adicionar ou listar tarefas.

## Funcionalidades
- Adicionar uma tarefa.
- Listar as tarefas pendentes.
